# QwenImg 简易启动指南

## 一次性设置（只需执行一次）

```bash
# 1. 运行环境设置脚本
chmod +x setup.sh
./setup.sh

# 2. 配置 API Key
export DASHSCOPE_API_KEY="sk-your-api-key-here"
```

## 日常启动

```bash
# 配置 API Key（如果还没配置）
export DASHSCOPE_API_KEY="sk-your-api-key-here"

# 启动服务
chmod +x run.sh
./run.sh
```

## 停止服务

```bash
./stop_dev.sh
```

## 访问

- 前端：http://localhost:3000
- 后端：http://localhost:8000
- API 文档：http://localhost:8000/api/docs

## 查看日志

```bash
# 后端日志
tail -f logs/backend.log

# 前端日志
tail -f logs/frontend.log
```

## 遇到问题？

### 完全重置

```bash
# 1. 停止服务
./stop_dev.sh

# 2. 重新设置环境
./setup.sh

# 3. 启动
export DASHSCOPE_API_KEY="your-key"
./run.sh
```
