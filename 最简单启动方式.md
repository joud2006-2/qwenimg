# 最简单的启动方式

## 第一次使用

### 1. 安装依赖

```bash
# 安装 Python 依赖
pip3 install --user uvicorn fastapi sqlalchemy aiosqlite pydantic pydantic-settings python-multipart websockets alembic python-jose passlib bcrypt python-dotenv dashscope Pillow requests

# 安装前端依赖
cd frontend
npm install
cd ..
```

### 2. 配置 API Key

```bash
# 方式1：环境变量
export DASHSCOPE_API_KEY="sk-your-api-key-here"

# 方式2：或创建 .env 文件
echo 'DASHSCOPE_API_KEY=sk-your-api-key-here' > .env
```

## 启动服务

### 终端1：启动后端

```bash
cd backend
python3 run.py
```

### 终端2：启动前端

```bash
cd frontend
npm run dev
```

## 访问

- 前端：http://localhost:3000
- 后端：http://localhost:8000

## 停止

在两个终端按 `Ctrl+C`

---

就这么简单！不需要虚拟环境，不需要复杂脚本。
